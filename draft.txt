- Create a chrome extension that adds filter: blur(ypx) to img tag
  - where y is retrieved from a slider in the extension's UI

- I want to
  - Be able to allow it for certain domains only
  - Be able to allow it globally
  - Same slider, no domain-specific sliders

- If global
  - Tab is going to listen for all messages
- If for certain domain
  - Tab is going to listen for messages on this domain
- On change
  - If change to global: we want all tabs to blur
    - Change to global
    - Send blur message
  - If change to domain-specific: we want some tabs to unblur
    - Before changing, send blur message at 0
    - change to domain-specific
    - Send blur message
